## Running Website Locally

In the terminal, run:

### `cd taylor-site`

Puts you in the correct directory to work
Then, in the project directory, run:

### `npm start`

Runs the app in development mode.


## Uploading to GitHub

In the terminal, run the following

###  `git add .`
### `git commit -m ""`

In the quotations, write a sentence describing the changes you made

### `git push`


## Using Multiple Devices

If you made changes on one device and want to update another to match, after pushing your changes on the first, 
run the following in your terminal on the second:

### `git fetch`
### `git pull`

Make sure anytime you make changes and are ready to give up for a while (real), push your changes to GitHub just in case
